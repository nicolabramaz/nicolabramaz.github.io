Site du développement web par Nicola Bramaz

Publié le 20 juillet 2017 HTML et CSS Nicola Bramaz

Comme vous le constatez, ici, mon but dans ce site est de montrer quels sont les développements de projets associatifs ou professionnels que j'ai mené à bien, afin de mettre en valeur mes capacités professionnelles dans le cadre de ma recherche d'un travail dans une entreprise. J’aime consacrer mon temps libre à ma passion, tout simplement. Le site a été développé dans un langage statique HTML/CSS.
